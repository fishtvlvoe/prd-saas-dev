# 案例：甲英英語補習班（v4 — 家長情緒解藥系統）

> 套用 `how/client-discovery-workflow.md` + `how/client-opportunity-mapping.md` + `how/emotional-value-design.md`

---

## 客戶背景

- 加盟佳音英語體系，**教務不是痛點**（總部標準化搞定）
- 真正痛點：**少子化下的招生困境**
- 目標：透過內容經營提高觸及與詢問度
- 商業企圖：做出**可複製、可推廣變現**的 AI 模板

---

## 四次方向校正記錄

| 版本 | 方向 | 為什麼錯 |
|------|------|---------|
| v1 | AI 教學工具（口說陪練 + 學習報告） | 教學不是甲英痛點，加盟總部搞定了 |
| v2 | 內容驅動招生引擎（成果影片） | 影片只是**爽點**不是痛點 |
| v3 | 每晚親子英文任務 | **摩擦力爆表** — 把陪讀責任丟回家長，違反「家長付錢為了外包陪讀」的本質 |
| v4 | **家長情緒解藥私域系統** | 解的是焦慮本身，不是英文進步 |

**核心教訓**：每次都在解「孩子的問題」，但**付錢的是家長**。家長不會為了解決孩子的英文買單，會為了解決自己的焦慮買單。

---

## 真正的痛點形狀（神經科學視角）

### 家長焦慮觸發鏈

```
孩子英文不好（觸發事件）
   ↓
投射「考不到好學校 / 未來輸給別人」（認知放大）
   ↓
跟其他家長比較 / 滑社群看到別人小孩優秀（社會比較）
   ↓
皮質醇 ↑ 失眠 煩躁 罪惡感（生理反應）
   ↓
找補習班 / 找方法 / 自責沒做好（行為輸出）
```

### 家長付錢的本質

```
表層：我要孩子英文進步
深層：我要不再焦慮、要感覺「我是好爸媽、我選對了」
   ↑
真正掏錢的層
```

**補習班賣的不是英文課，是家長焦慮的情緒解藥**。

---

## 方案核心：家長情緒解藥私域系統

### 用神經遞質視角設計四層處方

| 神經遞質 | 對應情緒 | 對應產品功能 |
|---------|---------|------------|
| **皮質醇 ↓** | 焦慮即時降低 | 焦慮時 LINE 私訊問 AI，立刻得到溫暖回應 + 案例 |
| **多巴胺 ↑** | 期待 / 小勝利 | 接送後收到「孩子今天的一句話」短語音 |
| **血清素 ↑** | 被認同 / 「我是好媽媽」 | 每週「你這週做對的 3 件事」清單 |
| **催產素 ↑** | 不孤單 / 戰友感 | 同年齡層家長私域配對、線下小聚 |

---

## MVP：家長 LINE 私域情緒系統

### 場景設計

**家長加入甲英 LINE 私域後**：

#### 每日（被動接收，零摩擦）
- **07:30** 推「今天的好媽媽小提醒」
  例：「昨天你忙到 11 點還記得幫小宇複習單字，這就是給孩子最好的身教 ❤️」
  → 血清素 ↑（被肯定）

- **接送後** 推「孩子今天說的一句英文」短語音 + 中文翻譯
  例：「小宇今天說：I want to be a scientist. 他說想當科學家喔！」
  → 多巴胺 ↑（小勝利）

#### 每週（強化）
- **週日晚** 推「你這週對孩子做對的 3 件事」
  → 血清素 ↑（持續肯定）

- **週日晚** 推「同類媽媽的故事」（同年齡 / 同煩惱）
  → 催產素 ↑（不孤單）

#### 焦慮時（即時止痛）
- 家長 LINE 私訊問「我家小孩○○怎麼辦」
- AI 草稿 + 老師審核 → 溫暖回應 + 真實案例
  → 皮質醇 ↓（即時止痛）

#### 私域社群（黏著）
- 「甲英家長戰友群」按年齡分群
- AI 配對「同年齡 + 同煩惱」家長
- 季度線下小聚（甲英舉辦）
  → 催產素 ↑（強連結）

## 對家長 / 甲英 / 孩子的負擔

| 對象 | 負擔 |
|------|------|
| 家長 | **零** — 被動接收 + 自願私訊 + 自願社群 |
| 甲英老師 | **低** — AI 草稿後老師審核 5 分鐘 |
| 孩子 | **零** — 完全不被打擾 |

對比 v3（親子任務）摩擦力炸裂，v4 摩擦力幾乎為零。

| 痛點處方 | 對應資產 | 改造幅度 |
|---------|---------|---------|

## 紅線守則（必須寫進產品憲法，違反 = 毀案）

從 `how/emotional-value-design.md` 帶過來：

1. ❗ **真實性**：所有推播內容必須來自「真實發生的事」，不造假
   - 孩子的金句必須是孩子真說過
   - 家長見證必須是真家長
   - 違反 = 詐騙

2. ❗ **底層空心檢查**：情緒服務建立在真實教學品質之上
   - 甲英的英文課必須真的有效
   - 紅線：教學品質低於佳音總部標準 → 拒絕導入此系統

3. ❗ **上癮 vs 安心離開**：每天只推 2 次（早晚）
   - 不做演算法 feed
   - 不做「下拉重整有新東西」的機制
   - 違反 = 變焦慮放大器

4. ❗ **私域不變焦慮放大器**：
   - 不公開孩子排名
   - 只展示「自家孩子相對自己的進步」
   - 過濾炫耀型內容（AI 審核）

5. ❗ **資料隱私**：
   - 家長私訊內容**不訓練 AI**
   - 不跨家長分享
   - 每個對話私密

## 三方案（情緒解藥視角）

### 方案 A：基礎私域（最輕）

- LineHub + AIRE：每日 2 推 + 每週 1 推
- 沒有 AI 私訊回應、沒有配對
- **適合**：先測水溫
- 月費基準：低

### 方案 B：完整情緒解藥系統（中）← **建議起點**

- 方案 A + AI 私訊回應（老師審核）+ 家長配對
- Voicebox 整合（孩子金句自動截取）
- **適合**：甲英當第一案例打磨，後續複製
- 收費：建置費低 + 月費 + 複製到其他補習班時甲英分潤

### 方案 C：全包招生引擎（重）

- 方案 B + 上游內容自動發 IG/FB + 試聽轉換漏斗 + 訂閱續報
- **適合**：作為「補習班招生模板」的旗艦案例
- 收費：建置費 + 月費 + 推廣分潤

## 為什麼這個有商業競爭力

| 維度 | 競爭力 | 說明 |
|------|-------|------|
| 競品沒做 | ✅ | 別家補習班都在賣「教學品質」，沒人賣「家長情緒服務」 |
| 黏著度 | ✅ | 私域 + 情緒依附 = 退訂痛 |
| 自帶傳播 | ✅ | 焦慮媽媽會拉同類媽媽進群 |
| 模板可複製 | ✅ | 換 logo 給其他補習班，前提：老闆有溫度 |
| 持續價值 | ✅ | 孩子畢業前家長都離不開（5-10 年生命週期） |
| AI 邊際成本 | ✅ | 一套內容引擎服務萬人 |

全部 6 條 ✅，遠超「值得做」的門檻（≥4 條）。

## 風險 / 反例

- ❌ **如果甲英教學品質沒到佳音標準** → 拒絕導入（紅線 2）
- ❌ **如果甲英老闆只想擠錢沒溫度** → 拒絕導入（情緒系統需要老闆理念支撐）
- ❌ **如果家長文化偏「鞭策孩子」非「陪伴孩子」** → 內容調性要調整，可能不適合
- ❌ **如果 Claude API 成本太高** → 用 Haiku 跑 80% 內容，Opus 只跑審核

## 從這個案例學到的（4 次跑偏的教訓）

- 教訓 1：**先做外部研究（Perplexity）再給方案**，不要憑印象猜痛點
- 教訓 2：**爽點不等於痛點**，影片是爽點，焦慮才是痛點
- 教訓 3：**摩擦力檢查必須做**，再好的方案家長不想動就是零
- 教訓 4：**付錢的人才是真客戶**，家長才是客戶，孩子是商品（殘酷但真實）

這 4 個教訓會寫進 `how/client-opportunity-mapping.md` 變正式判別規則。
