# 案例：甲英英語補習班（2026-05）

> 套用 `how/client-discovery-workflow.md` + `how/client-opportunity-mapping.md` 的真實案例。

---

## 客戶背景

- 加盟佳音英語體系，**教務不是痛點**（總部標準化搞定）
- 真正痛點：**行銷招生**，少子化下傳統招生方式不夠用
- 目標：透過內容經營提高觸及與詢問度
- 商業企圖：希望能做出**可複製、可推廣變現**的 AI 模板

---

## 第一次判斷的錯誤（重要校正）

最初我假設痛點在教學端（口說回饋慢、家長看不見成效）。
Fish 校正：**這方向錯了**。

| 第一次假設 | 校正後 |
|------------|--------|
| 痛點在教學 | 痛點在行銷招生 |
| 解法 = AI 陪練 | 解法 = 內容驅動的招生引擎 |
| 賣家長：孩子成績進步 | 賣家長：孩子敢開口、不排斥 |

教訓：**外部研究（Perplexity）必須先做，不要先猜方向**。

---

## Perplexity 研究結論（12 歲以下家長真正在意）

家長買單的不是「英文很強」，而是這三件事：

```
1. 孩子願意學（不排斥、不抗拒）
2. 家長能跟得上（爸媽英文不好也能陪）
3. 結果看得見（不是分數，是「敢開口」的證據）
```

家長最焦慮的痛點清單（從 Dcard / Threads 公開討論抓出）：
- 怕孩子排斥英文 / 學了不敢說
- 怕發音學歪
- 爸媽自己英文不好，沒辦法陪
- 孩子坐不住、專注力差
- 花錢看不到效果
- 不知道幾歲開始好
- 選擇太多，不知道選哪種

## 關鍵洞察：找到「自然傳播」的爆點

行銷的最強招式不是「補習班自己打廣告」，是「**讓家長自己幫你傳**」。

**爆點公式**：

```
孩子開口錄影 → AI 剪成 30 秒成果影片 → 家長收到 → 家長轉發 LINE 群
   ↓
其他家長看到 → 詢問補習班 → 試聽 → 報名
```

這個迴圈一氣呵成：
- **孩子端**：看到自己被剪成影片 → 有成就感 → 願意再來
- **家長端**：收到孩子說英文的影片，比成績單強 10 倍 → 自然轉發
- **補習班端**：影片成為招生素材（家長同意後）+ 自帶傳播 → 招生成本下降

這比「家長報告 PDF」強 100 倍，因為**影片可以轉發，PDF 不行**。

---

## 三方案（行銷招生視角）

### 方案 A：內容工具租賃（最輕）

包月給甲英用：
- AIRE 生成招生內容（IG 短影音腳本、家長焦慮型貼文）
- LineHub 詢問自動回應
- BuyGo+1 試聽預約表單

**適合**：先測水溫，甲英自己操作
**月費基準**：低（SaaS 訂閱）

### 方案 B：共同開發成果影片引擎（中度）

用甲英當第一個案例打磨「補習班招生 AI 套組」：
- Voicebox + MOLTOS 整合：孩子錄音 → AI 剪 30 秒成果影片
- LineHub 自動派發給家長
- AIRE 自動生成轉發文案模板

**適合**：甲英願意當白老鼠，我們吸收前期開發成本，後續複製到其他補習班
**收費**：建置費低 + 月費 + 後續複製到其他補習班時甲英分潤

### 方案 C：全包招生引擎（最重）

從觸及 → 詢問 → 試聽 → 報名 → 學習 → 續報全鏈路：
- 上游：AI 內容生成 + 自動發 IG / FB / LINE
- 中游：詢問自動回應 + 試聽預約
- 下游：學習中的開口成果影片 → 自動回到上游當招生素材

**適合**：作為「補習班招生模板」的完整實驗
**收費**：建置費 + 月費，可作為對外推廣的旗艦案例

## MVP 建議：成果影片引擎（方案 B 起點）

**核心功能**：孩子在補習班錄一段英文 → AI 剪成 30 秒影片 + 家長中文說明 → 自動傳到家長 LINE。

### 為什麼選這個

| 判準 | 滿足度 |
|------|-------|
| 家長 7 天內感受到價值 | ✅ 第一週就收到孩子說英文影片 |
| 老師零學習成本 | ✅ 老師按一顆錄音鈕，其他自動跑 |
| 解最核心痛點 | ✅「敢開口」+「看見成果」一次解 |
| 自帶傳播性 | ✅ 影片家長會自發轉發 |
| 可複製給其他補習班 | ✅ 每家補習班都需要 |
| 改造幅度可控 | ✅ Voicebox + LineHub + AIRE 都現成 |

### 開發範圍

- Voicebox 整合英文 ASR + 簡單發音評分
- 影片自動剪輯模板（孩子原音 + 字幕 + AI 老師鼓勵聲音）
- LineHub 自動派發 + 中文家長說明文案
- 後台老師介面：選孩子 → 按錄音 → 影片自動產出

### 時程估算

2-3 週可以做出可用版本（既有元件多，主要是整合）。

## 反例 / 風險

- ❌ **如果甲英只想要「教學工具」**：方向就錯了，他真正的痛點是招生，要回到原點
- ❌ **如果家長不願意孩子被錄影**：需要隱私同意機制，並提供「只給家長看，不公開」選項
- ❌ **如果 Voicebox 對兒童英文 ASR 準度不夠**：先用「老師人工修字幕」過渡，不要等技術完美才上線
- ❌ **如果模板化過度，失去甲英的特色**：方案 B 階段要保留客製空間，方案 C 才模板化
