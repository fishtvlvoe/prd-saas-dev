# 先找對的人，不是找對的問題（Right People, Not Right Problems）

這個檔案解決的問題：銷售/顧問案常陷入「客戶有痛點 → 我有方案 → 我賣 → 死案」的迴圈。根因不是方案不好，是**對方沒能力或沒意願執行**。這份方法論定義「對的人」的篩選機制。

---

## 核心命題

> 「有問題的客戶」不等於「會解決問題的客戶」。
>
> 銷售的本質不是「把方案賣出去」，是「**篩選出能執行的人**」。

簽錯人 = 案子做不下去 = 客戶不滿 = 毀牌。
篩對人 = 給方向他就跑 = 案例做出來 = 自帶口碑。

---

## 三型客戶分類

| 類型 | 特徵 | 是否值得簽 |
|------|------|-----------|
| **問題型** | 知道有問題、想解決，但**沒人、沒時間、沒決心** | ❌ 簽下來是地獄 |
| **方案型** | 已經有方向，缺工具或專業協助 | ⚠️ 看適配度 |
| **行動型** | 有意願 + 有能力 + 有團隊 + 能下決策 | ✅ 真客戶 |

問題型客戶最危險 — 他們最容易上鉤（因為焦慮），但簽下來執行不下去，最後罵的是你。

---

## 對的人的 6 個訊號

每次提案前過一次。少於 4 個 → 不簽。

| 訊號 | 怎麼觀察 |
|------|---------|
| 1. **有時間** | 願意給你 30 分鐘以上對談、會主動約第二次 |
| 2. **有人執行** | 有員工或合作夥伴可以承接，不是「老闆 1 人撐」 |
| 3. **員工願意配合** | 老闆說「我團隊會挺我」，不是「我得說服他們」 |
| 4. **決策魄力** | 一週內能拍板，不是「我得跟太太/股東/總部商量」 |
| 5. **預算意識** | 對「投資 vs 成本」分得清楚，不是只問「最便宜多少」 |
| 6. **自主性** | 給方向後會自己想、自己研究，不是「你來幫我想就好」 |

---

## 篩選法：給方向，不給方案

### Step 1：聊 + 觀察（讓他自己說痛）

問開放式問題：
- 「你最近最頭痛的事是什麼？」
- 「你覺得 X 問題對你影響大不大？」
- 「現在你是怎麼處理的？」
- 「你有沒有想過 Y？」

**禁止**：自己描述他的痛點再讓他點頭（這叫引導，不叫篩選）。

### Step 2：點一句金句（測試共鳴）

給一句**反直覺的洞察**，不是「我有方案」。

範例：
- 「補習班賣的不是英文課，是家長焦慮的情緒解藥。」
- 「你以為員工要的是加薪，但他們真正想要的是被看見。」
- 「你不缺客戶，缺的是不適合你的客戶被擋掉。」

**觀察反應**：

| 反應 | 解讀 | 動作 |
|------|------|------|
| 眼睛亮 + 追問 | 對的人 | 繼續講 |
| 點頭附和但無情緒 | 客套 | 停 |
| 皺眉懷疑 | 不適配 | 換話題 |
| 沉默思考 | 在內化 | 等他開口 |
| 開始講自己的故事 | 高度共鳴 | 多聽少說 |

### Step 3：給「自己做」的選項（測試行動力）

關鍵句型：
> 「這個方向**不一定要找我**。你可以自己試試看 — 比方 [具體輕量做法]。如果你想試，我可以給你方法。」

**這句話是金鑰**，會分流出三種人：

| 對方回應 | 真意 | 你的動作 |
|---------|------|---------|
| 「我自己做就好」 | 行動型，但不需要你 | 給方法、保持聯絡（未來可能合作） |
| 「聽起來不錯，但我沒人做 / 沒時間」 | **真客戶**，主動上門 | 開始談合作 |
| 「再想想」 | 不是時候 | 不勉強，3 個月後再聯絡 |
| 「你能不能直接幫我做」 | 問題型（要外包但沒準備好） | 進一步測試他的決心 |
| 「方法直接給我」然後失聯 | 白嫖 | 不可惜，篩走 |

## 銷售心法：等對方拉，不是你推

```
推（硬賣）= 你抓著對方袖子說「來啊來簽啊」
拉（被動）= 你站在門口說「方向在這，要進來找我」
```

**推**的後果：
- 簽到一堆問題型客戶
- 案子做不下去
- 自己被消耗

**拉**的後果：
- 篩出真的會買的人
- 案例做出來
- 自帶口碑、後續客戶自己上門

INFJ / 一人公司特別適合「拉」 — 你體力撐不住「推」的消耗。

---

## 問題型客戶會帶來什麼地獄（反例警示）

簽下問題型客戶，會出現以下症狀：

1. **無限延後**：「下個月再開始」「再等等我老婆出國回來」
2. **執行斷層**：方案給了沒人做，回頭問你「能不能幫我執行」
3. **永遠不滿**：因為他原本就無能力評估，永遠覺得「沒看到效果」
4. **議價無底洞**：每個階段都要砍價，你不降他就拖
5. **責任轉嫁**：失敗了怪你，成功了說是他的本事
6. **毀牌風險**：他到處說你的壞話，你的下一個客戶聽到就跑

**經驗法則**：直覺覺得「這人有點怪」、「這案子怎麼一直簽不下來」、「我得追著他催」 → 99% 是問題型，立刻退場。
